# Cryptomania

This website lists all the crypto coins,details of that coin,news around crypto etc.

## Tech Stack

**Client:** React,Ant-Design

**API:** Rapid Api(coinranking)

## Demo

[Live-Project](https://jade-quokka-38c23e.netlify.app//)

## Features

- Crypo coin details
- Top 10 Crypto
- News about crypto
- Mobile responsive
- Coin price graph
